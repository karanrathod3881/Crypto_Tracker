<h1>Crypto Price Tracker</h1>

A simple React-based web application that displays real-time cryptocurrency prices using the CoinGecko API.

## Features
- Real-time price updates
- Search cryptocurrencies
- Interactive price charts (last 7 days)
- Sorting (Low to High, High to Low)
- User authentication using Firebase
- Optimized performance using React hooks

## Tech Stack
- React.js
- Firebase Authentication
- Chart.js
- CoinGecko API

## Run Locally

```bash
npm install
npm start
